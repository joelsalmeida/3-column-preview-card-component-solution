### What I learned

It is possible to use blend modes in CSS.
In this challenge, I used "screen" mode to create transparency.

```html
<button class= "button-transparent">My Button</button>
```
```css
.button-transparent {
    color: black;
    mix-blend-mode: screen;
}
```
